What's it?

我们在给顺丰的消息推送服务器做一个简单的性能测试，证明单台服务器可以支持10万个TCP长连接。 如果只依靠一台两台客户端来连，每个TCP连接不能太快，否则会被服务器拒绝掉。昨天花了将近1天的时候，才 为了快速达到上万甚至十万个连接，我们需要大家帮忙点击上面的连接即可。

How it works?

点击上面的连接之后，浏览器会下载一个WebStart文件，点击该文件，我们会在你机器上启动一个Java小应用，来连我们的服务器。连接的频率是每10秒1个连接，上限是1000个连接。所占的CPU和内存都很有限，因此对你的正常使用不会造成任何影响。
